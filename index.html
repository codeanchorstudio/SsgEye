HOW IT WORKS

어떻게
동작하나요?

복잡한 설정 없이 설치 후 바로 사용할 수 있습니다.

01
환자 정보 감지
접수 프로그램에서 환자를 조회하고 엔터 또는 시작 버튼을 누릅니다.
02
자동 추출
성명, 차트번호, 생년월일, 성별을 접수 프로그램에서 자동으로 읽어옵니다.
03
장비 분배
실행 중인 장비 프로그램에 순서대로 환자 정보를 자동 입력합니다.
04
완료 확인
각 장비별 성공 / 실패 / 건너뜀 상태를 실시간으로 확인합니다.
