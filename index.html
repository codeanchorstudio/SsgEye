Installation

설치 방법

아래 순서대로 설치를 진행해 주세요.

1
다운로드
위 다운로드 버튼을 클릭하여 MediSync.exe 또는 MediView.exe 파일을 저장합니다.
2
관리자 권한으로 실행
다운로드한 .exe 파일을 우클릭 → 관리자 권한으로 실행을 선택합니다. (Windows Defender 경고 시 "추가 정보 → 실행" 클릭)
3
초기 설정 확인
프로그램 첫 실행 시 대상 프로그램 경로 설정이 필요할 수 있습니다. 설정 화면에서 각 검사 프로그램 경로를 확인해 주세요.
4
사용 시작
설치 완료 후 기존 접수·진료 업무 흐름 그대로 사용하시면 됩니다. MediSync는 대시보드 버튼, MediView는 NixPen Chart 조작 시 자동 작동합니다.
FAQ

자주 묻는 질문

MediSync와 MediView를 동시에 사용할 수 있나요?
네, 두 프로그램은 독립적으로 동작하므로 동시에 실행해도 충돌이 없습니다. 원무 접수 시 MediSync, 진료 차트 조작 시 MediView가 각각 자동화를 담당합니다.
Windows Defender가 프로그램을 차단하면 어떻게 하나요?
"Windows에서 PC를 보호했습니다" 창이 뜨면 추가 정보를 클릭한 뒤 실행 버튼을 눌러주세요. 사내 전용 프로그램이므로 코드 서명 인증서가 없어 발생하는 정상적인 경고입니다.
검사 프로그램이 업데이트되면 자동화가 깨지나요?
검사 프로그램의 UI 구조가 크게 변경될 경우 재설정이 필요할 수 있습니다. 이 경우 담당자에게 문의해 주시면 업데이트된 버전을 제공해 드립니다.
MediView 사용을 위해 NixPen Chart 설정이 필요한가요?
별도의 NixPen Chart 설정 변경은 필요하지 않습니다. MediView는 키보드·마우스 이벤트를 백그라운드에서 모니터링하는 방식으로 동작하므로 기존 사용 방식을 유지하시면 됩니다.
환자 데이터가 외부로 전송되나요?
아니요. 두 프로그램 모두 인터넷 통신 없이 로컬 PC 내에서만 동작합니다. 환자 정보는 원내 네트워크 범위 내에서만 처리됩니다.
